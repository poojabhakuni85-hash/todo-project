# React Todo App

A simple and responsive Todo List application built using React Hooks.

## Features

- Add Todo using button
- Add Todo using Enter key
- Delete Todo
- Clean and responsive UI

## Technologies Used

- React JS
- JavaScript
- HTML
- CSS
